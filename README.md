# UPI Transaction Analytics Dashboard

A full-stack fintech portfolio project simulating analytics systems used by PhonePe, Google Pay, and Paytm.

## Features
- Fraud detection & intelligence
- Spending behaviour analysis
- Interactive charts (Recharts)
- Transaction explorer with 25,000 synthetic UPI records

## Stack
- **Frontend:** React + Vite, Recharts, TanStack Query, Wouter
- **Backend:** Express 5, Node.js 24, TypeScript
- **Database:** PostgreSQL + Drizzle ORM
- **Monorepo:** pnpm workspaces
